# t-ui

## 安装依赖
```shell
npm install
```


## 本地运行src中组件示例
```shell
npm run serve
```
## 本地运行vuepress中组件文档
```shell
npm run docs:dev
注：如果安装依赖后console报core.js错，需要下载2.**版本中把library文件夹移动到此项目node_modules中core.js中
```

### 项目打包
```
npm run build
```
